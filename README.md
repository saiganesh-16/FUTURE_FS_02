# 💼 Client Lead Management System (Mini CRM)

A full-stack CRM dashboard built for businesses to track, manage, and update client leads from their website. This project was developed as Task 02 for the Future Interns Full Stack Web Development Internship.

## 🚀 Features
- **Full CRUD Operations:** Add, view, update, and delete leads.
- **Status Tracking:** Update lead status (New → Contacted → Converted).
- **Responsive Dashboard:** Modern, clean UI built with React.
- **RESTful API:** Node.js/Express backend communicating with a MongoDB database.

## 💻 Tech Stack
- **Frontend:** React.js, Vite, Axios, CSS3
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas

## 🛠️ Setup Instructions

### 1. Clone the repository
\`\`\`bash
git clone https://github.com/saiganesh-16/FUTURE_FS_02.git
cd FUTURE_FS_02
\`\`\`

### 2. Setup the Backend
\`\`\`bash
cd backend
npm install
node server.js
\`\`\`
*(The backend will run on http://localhost:5000)*

### 3. Setup the Frontend
Open a new terminal window:
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
*(The frontend will run on http://localhost:5173)*
